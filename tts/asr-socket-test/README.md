# ASR Socket Test Project

このプロジェクトは、自動音声認識（ASR）システムのモックサーバーとクライアントの通信をテストするためのものです。以下にプロジェクトの概要、セットアップ手順、使用方法を示します。

## プロジェクトの概要

- `src/asr_mock_server.py`: ASRモックサーバーを実装し、クライアントからの接続を受け入れ、テスト用のデータを送信します。
- `src/socket_client_check.py`: モックサーバーに接続し、サーバーからの応答を受け取り、データの送受信が正しく行われるかを確認します。
- `tests/test_protocol_parsing.py`: モックサーバーとクライアントの通信が正しく行われているかを確認するためのユニットテストを含みます。


## 使用方法

1. モックサーバーを起動します。
   ```
   python src/asr_mock_server.py
   ```

2. 別のターミナルでクライアントを実行します。
   ```
   python src/socket_client_check.py
   ```

3. サーバーとクライアントの通信が正しく行われることを確認します。
